# Saint Thomas Institute (STI) Website

A modern school website for Saint Thomas Institute with a Navy Blue & Gold color scheme.

## Files Included

- **index.html** - Main HTML file with complete website structure
- **style.css** - Complete styling with Navy Blue (#1e3a8a) and Gold (#f59e0b) color scheme
- **campus.jpg** - Placeholder for campus/building image (needs to be replaced with actual image)

## Features

- Responsive design for all screen sizes
- Smooth scroll animations
- Interactive navigation with mobile hamburger menu
- Contact form with submission feedback
- Modern glassmorphism effects
- Gradient backgrounds and shadows
- Scroll reveal animations
